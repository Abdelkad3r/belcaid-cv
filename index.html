Statement

A Senior Security Engineer with deep experience securing enterprise environments — from web applications and cloud workloads to operational technology on the factory floor. I bridge offensive testing, defensive engineering, and governance, because resilient security needs all three.

I've led Active Directory remediation at scale, driven patching campaigns across thousands of vulnerabilities, and designed network segmentation isolating OT from IT in industrial environments. I treat every system as something an adversary will eventually probe — and I build accordingly.
